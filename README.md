# logical_peapod
PEAPOD logical expansion

## Build & Run
Once per terminal session, from main dir, run:
```console
  . classpath
```
To compile:
```console
  make
```
To clean:
```console
  make clean
```
To run client:
```console
  ./runClient.sh [port] -ip [ipadd] [other args]
```
To run server:
```console
  ./runServer.sh [port]
```
To kill server:
```console
  ./killServer.sh
```
